## 3. Teknik Özellikler

### 3.1 Blockchain Altyapısı
- **Ağ:** Solana (SOL)
- **Token Standardı:** SPL (Solana Program Library)
- **Hız:** ~400ms blok süresi
- **Maliyet:** Düşük işlem ücretleri (<$0.01)

### 3.2 Tokenomics
TUNAMI, topluluk odaklı ve adil bir dağılım modelini benimser.

- **Toplam Arz:** 1,000,000,000 (1 Milyar) $TUNAMI
- **Vergi (Tax):** %0 Alış / %0 Satış
- **Likidite:** %100 Kilitli (Burned)
- **Mint Yetkisi:** İptal Edildi (Revoked)

Yatırımcı güvenliği projenin temel önceliğidir.
